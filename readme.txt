WordPress dashboard widget recharts to show data

== Description ==

Recharts is a great plugin which show a WordPress dashboard widget for the last 7, 15 or 30 days using graph. It is showed on WordPress dashboard as a widget.

== Installation ==

Here is the process how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Do I need to do anything to show the widget? =

No, it will show the graph automatically based on the date from the past data for 7, 15 or 30 days.
